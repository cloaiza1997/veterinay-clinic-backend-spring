## Ejecución

- Limpiar compilado

  ```
  ./gradlew clean
  ```

- Compilar

  ```
  ./gradlew build
  ```

- Ubicación del compilado

  ```
  build\libs\veterinary.clinic-0.0.1-SNAPSHOT.jar
  ```

- Ejecutar compilado

  ```
  java -jar veterinary.clinic-0.0.1-SNAPSHOT.jar
  ```
